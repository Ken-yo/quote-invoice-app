# 見積・請求書作成アプリ

## 概要

顧客情報の登録、見積書・請求書の作成、明細入力、金額計算、Excel出力を行う業務アプリです。

## 想定課題

小規模事業者や副業エンジニアが、見積書や請求書をExcelで手作業作成しているケースを想定しています。  
手作業では、金額計算ミス、帳票番号の管理漏れ、過去履歴の確認しづらさが課題になります。

## 解決策

本アプリでは、顧客情報と明細を入力するだけで、小計・消費税・合計金額を自動計算し、Excel形式の見積書・請求書を作成できます。

## 主な機能

- 顧客情報登録
- 見積書作成
- 請求書作成
- 明細入力
- 小計・消費税・合計金額の自動計算
- SQLiteへの保存
- 作成履歴表示
- Excel帳票出力
- 顧客情報の手入力登録
- CSV/Excelファイルによる顧客情報の一括登録
- アップロード内容のプレビュー
- 会社名重複時のスキップ処理
- 顧客ごとの品目マスタ登録
- 見積・請求書作成時の品目リスト選択
- 品目選択時の標準数量・単価自動反映
- 顧客別の単価管理

## 使用技術

- Python
- Streamlit
- SQLite
- openpyxl
- pandas
- pytest

## 工夫した点

- 画面、DB処理、計算ロジックをファイル分割した
- 税率を変更できるようにした
- 見積書と請求書で帳票番号の接頭辞を分けた
- Excel出力を業務帳票として使いやすい形に整えた
- 計算ロジックにテストコードを追加した
- 顧客情報を手入力だけでなくCSV/Excelアップロードでも登録できるようにした
- アップロード前に内容をプレビュー表示し、登録前に確認できるようにした
- 会社名が既に登録済みの場合はスキップし、重複登録を防止した
- CSVの文字コードはUTF-8-SIGとCP932の両方に対応した
- 顧客ごとに品目マスタを持たせ、見積作成時に選択できるようにした
- 顧客別に単価を管理できるため、同じ作業内容でも顧客ごとに異なる単価設定に対応できる
- 品目選択後も数量や単価を編集できるようにし、実務上の調整に対応した
